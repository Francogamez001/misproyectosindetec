Proyecto 1: Calculadora en JavaScript

Descripción del proyecto

Este proyecto consiste en una calculadora básica desarrollada con HTML, CSS y JavaScript. Permite realizar operaciones simples como suma, resta, multiplicación y división directamente en el navegador, sin necesidad de recargar la página.

Vista del proyecto

La interfaz está pensada para ser sencilla y clara, con botones para cada operación y una pantalla donde se muestran los números y el resultado. Este proyecto me permitió practicar el manejo del DOM, eventos de clic y el uso de funciones en JavaScript.
