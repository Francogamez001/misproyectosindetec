Proyecto 3: auto impreso en 3D

Descripción del proyecto

En este proyecto diseñamos el modelo 3D a escala de un soporte para los componentes de un auto hecho con una placa arduino uno, en donde se utilizan dos motores para poder girar las cuatro ruedas del auto,un puente H para conectar los motores a la placa.

Vista del proyecto

En este proyecto diseñamos un modelo 3D a escala que actúa como soporte estructural para los distintos componentes electrónicos de un vehículo prototipo controlado mediante una placa Arduino Uno. El diseño permite montar de manera ordenada y segura cada módulo necesario para el funcionamiento del vehículo, asegurando la correcta distribución del peso y la estabilidad del conjunto.
