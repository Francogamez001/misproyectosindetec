Proyectos

Proyecto 1: Leds cambiando cada segundo

En este proyecto, se utilizo una placa de arduino uno para prender y apagar dos leds cada un segundo. mientras uno de los leds se enciende, el otro permanece apagado.

Ver proyecto

Proyecto 2: Led con sensor infrarrojo

En este proyecto mediante un sensor infrarrojo y un control, se puede encender un led conectado a una placa de arduino uno. Cuando se presiona el boton de encendido, el led se prende, cuando se presiona el boton de pausa, el led se apaga.

Ver proyecto

Proyecto 3: Auto impreso en 3D

En este proyecto diseñamos el modelo 3D a escala de un soporte para los componentes de un auto hecho con una placa arduino uno, en donde se utilizan dos motores para poder girar las cuatro ruedas del auto,un puente H para conectar los motores a la placa..
